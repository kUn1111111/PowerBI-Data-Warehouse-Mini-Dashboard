# PowerBI Data Warehouse Mini + Dashboard

## Project Overview
This project showcases a mini Data Warehouse solution complete with a Power BI Dashboard. It serves as a demonstration of foundational data modeling, ETL logic, and interactive analytics using a sample sales dataset, making it ideal for anyone seeking hands-on Business Intelligence (BI) practice.

## Key Features
- **Star Schema Data Model:** Includes fact and dimension tables for BI reporting.
- **Power BI Dashboard:** Offers interactive visualizations—KPIs, charts, trends, and more.
- **Sample Metrics:** Total Sales, Sales by Region, Product Performance, and others.
- **Beginner-Friendly:** Designed for learning Power BI and data warehousing basics.


## Technologies Used
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)** for calculated measures
- **Sample datasets (CSV/Excel)** for practice

## How to Use

1. **Clone the repository:**
    ```sh
    git clone https://github.com/yourusername/PowerBI_DataWarehouse_Mini.git
    ```

2. **Open Power BI Project:**
   - Open the Power BI Desktop file located in the `PowerBI_Dashboard/` folder.

3. **Connect Data:**
   - Ensure Power BI links to the sample files within the `Data/` folder.

4. **Explore & Customize:**
   - Review dashboards, KPIs, and visualizations. Feel free to tailor the reports or underlying data.
